# MCP2VMCP
[mocopi](https://www.sony.jp/mocopi/)の姿勢情報を、[VMCProtocol](https://sh-akira.github.io/VirtualMotionCaptureProtocol/)で送信するソフトウェア。  
非常に簡易的な機能ですが、VRM0とVRM1の両方に対応しています。
